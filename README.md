# SNAP-COIN-BOT

## REQUIREMENTS

1. python
2. discord bot token

## Install

```pip install -r requirements.txt```

```make run```

## ENV

Create .env and set required fields

```.env
DISCORD_TOKEN="your_discord_bot"
SECRET_KEY="secret_token_here"
FAUCET_LINK ="https://faucet.snap-coin.net:9999/withdraw-faucet-dev"
```
